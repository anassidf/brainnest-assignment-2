13. What is the difference between a block element and an inline element?

  • block element: it takes the full width of its container and take a new line
  • inline element: it present in the same line as its container and it take the width of the conent

14. What is the difference between an inline element and an inline-block element?

  • inline-block: the width and the height can be specified , the top and bottom margins/paddings are respected
  • inline element: the top and bottom margins/paddings are not respected

19. What’s the difference between a flex container and a flex item?

the flex container is the parent and the container that we will specify the styles of the flex box in bu the flex item are the children of the flex container and the elements that will be positioned in the flex container

23. What is the difference between justify-content and align-items?

  • justify-content: align the items horizontally
  • align-items: align the items vertically

24. How do you use flexbox to completely center a div inside a flex container?

25. What’s the difference between justify-content: space-between and justify-content: space-around?
